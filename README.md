# Manual API Testing Project

## Project Overview

This project demonstrates manual API testing of the Evershop e-commerce application.
website Link : https://demo.evershop.io/

The project includes:

* Postman Collection
* API Test Cases (Excel)

## Tools Used

* Postman
* Microsoft Excel
* GitHub

## Test Coverage

### Search Module

* Search product using keyword

### Product Module

* View product details
* View customized product details

### Cart Module

* Add item to cart
* View cart
* Increase item quantity
* Decrease item quantity
* Delete cart item
* Verify cart after update

### Coupon Module

* Apply invalid coupon

### Checkout Module

* View checkout page

## API Endpoints Tested

| Module               | Method |
| -------------------- | ------ |
| Search Item          | GET    |
| View Item            | GET    |
| View Customized Item | GET    |
| Add To Cart          | POST   |
| View Cart            | GET    |
| Increase Quantity    | PATCH  |
| Decrease Quantity    | PATCH  |
| Apply Coupon         | POST   |
| Checkout             | GET    |
| Delete Cart Item     | DELETE |
